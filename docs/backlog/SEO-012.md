# SEO-012: Interactive Container Planner Tool

**Priority:** P3
**Effort:** XL (8+ hours)
**Status:** ready
**Needs Assistance:** No (but significant dev effort)

## Description
Create an interactive tool for customers to plan container modifications. The Container Guy has "PlanMyCan" - a 2D modification planner. This would be a significant differentiator and lead generation tool.

## Feature Concept

### Basic Version (MVP)
- Select container size (20ft, 40ft, etc.)
- Visual representation of container
- Click to add modifications:
  - Doors (placement options)
  - Windows (sizes)
  - Electrical package
  - Insulation
  - Ventilation
- Generate summary/quote request

### Advanced Version
- 3D visualization
- Drag-and-drop placement
- Real-time pricing estimates
- Save/share designs
- PDF export

## Technical Approach

### MVP Stack
- Vanilla JavaScript or React component
- SVG-based container visualization
- Form-based modification selection
- Generates quote request with selections

### Data Structure
```javascript
{
  containerSize: "40ft-standard",
  modifications: [
    { type: "man-door", position: "front-left" },
    { type: "window", position: "side-right", size: "3x3" },
    { type: "electrical", package: "basic" }
  ],
  notes: "Need by March 2026"
}
```

## User Flow
1. Land on `/container-planner` page
2. Select base container
3. Add modifications via UI
4. See visual update
5. Submit for quote
6. Lead captured with detailed requirements

## SEO Benefits
- Unique interactive content
- Time on site increase
- Lead generation
- Shareable tool (backlink potential)
- Targets "custom container" searches
