# SEO-001: Add FAQ Schema to Key Pages

**Priority:** P1
**Effort:** S (1-2 hours)
**Status:** ready
**Needs Assistance:** No

## Description
Add FAQPage structured data schema to key landing pages to enable rich snippets in Google search results. Competitors like ATS Containers and Coast Containers use this effectively.

## Pages to Add FAQ Schema
- `/shipping-containers` - Sales page
- `/storage-container-rentals` - Rentals page
- `/on-site-storage-rentals` - On-site storage page
- `/shipping-container-guide` - Guide page (already has HowTo, add FAQ)
- City landing pages (all 5)

## FAQ Content to Create
Each page should have 3-5 relevant FAQs. Examples:

**Sales Page:**
- What sizes of shipping containers do you sell?
- What's the difference between new and used containers?
- Do you deliver shipping containers?
- How much does a shipping container cost in Saskatchewan?

**Rentals Page:**
- What are the rental terms for storage containers?
- Can I rent a container for just one month?
- Do you offer rent-to-own options?
- What's included in the rental price?

## Implementation
1. Create `FAQSchema.astro` component
2. Add FAQ content to each page's frontmatter or as props
3. Render schema in `<head>`

## Acceptance Criteria
- [ ] FAQSchema component created
- [ ] FAQ content added to 5+ pages
- [ ] Schema validates in Google Rich Results Test
- [ ] Build passes

- ATS Containers uses FAQ schema on service pages
- Coast Containers has detailed FAQ sections
