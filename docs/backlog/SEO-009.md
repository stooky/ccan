# SEO-009: Staff/Founder Profile Section

**Priority:** P2
**Effort:** S (1-2 hours)
**Status:** blocked
**Needs Assistance:** Yes - need photo and bio from Sam

## Description
Add a personal founder/staff profile section. BigSteelBox does this effectively with store manager profiles including photos. This adds trust and humanizes the business.

## Content Needed from Owner
1. **Photo** - Professional headshot or on-site photo
2. **Bio** - 2-3 paragraphs about:
   - Background and experience
   - Why started C-Can Sam
   - What makes the business different
   - Personal connection to Saskatchewan
3. **Optional:** Team photos if applicable

## Implementation (once content gathered)

### About Page Enhancement
Add "Meet Sam" or "Our Story" section with:
- Photo (optimized for web)
- Personal bio
- Quote or mission statement
- Optional: Video introduction

### Homepage Addition
Consider small "Local & Family-Owned" callout with photo

### Schema Markup
Add Person schema linked to LocalBusiness:
```json
{
  "@type": "Person",
  "name": "Sam",
  "jobTitle": "Founder",
  "worksFor": { "@type": "LocalBusiness", "name": "C-Can Sam" }
}
```

## Benefits
- Builds trust with local customers
- Differentiates from faceless corporations
- Supports E-E-A-T (Experience, Expertise, Authority, Trust) signals
- Makes the brand memorable

- [ ] Photo obtained from owner
