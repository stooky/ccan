# SMB Website Verticals

A taxonomy of high-demand, lucrative SMB categories for website generation. These verticals are selected based on:
- High customer lifetime value
- Strong local SEO potential
- Service-based (recurring revenue opportunities)
- Typically underserved by quality web presence

## Implementation Strategy

### Phase 1: Template Variants
Create vertical-specific configurations:
```
src/
├── verticals/
│   ├── home-services/
│   │   ├── config.ts      # Industry-specific defaults
│   │   ├── services.ts    # Common services list
│   │   ├── content/       # Sample content
│   │   └── components/    # Special components (emergency banner, etc.)
│   ├── legal/
│   ├── medical/
│   └── ...
```

### Phase 2: Vertical-Specific Components

**Home Services:**
- Emergency banner with phone number
- Service area map
- Financing calculator
- Maintenance plan signup
- Before/after gallery

**Legal:**
- Practice areas grid
- Case results (with disclaimers)
- Attorney profiles
- Free consultation CTA
- Legal resources/blog

**Medical/Dental:**
- Insurance accepted list
- New patient forms
- Provider profiles
- Patient testimonials (HIPAA compliant)
- Online scheduling integration

**Construction:**
- Project portfolio with filters
- Process timeline
- Licensing/insurance badges
- Financing partners
- Design inspiration gallery

### Phase 3: Content Templates

Pre-written, customizable content for:
- Homepage headlines and copy
- Service descriptions
- About page frameworks
- FAQ content
- Blog post templates
- Email sequences

### Phase 4: Integration Presets

Recommended integrations by vertical:
- **Home Services:** ServiceTitan, Housecall Pro, Jobber
- **Legal:** Clio, LawPay, Calendly
- **Medical:** Jane App, SimplePractice, Zocdoc
- **Construction:** Buildertrend, CoConstruct, Houzz
- **Restaurants:** Toast, Square, OpenTable
